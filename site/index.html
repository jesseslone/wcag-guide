Accessibility compliance, together

You bring the expertise. Your AI agent brings the throughput. WCAG-Guide is an open-source scanner and remediation tracker where humans and AI agents work side by side to ship accessible websites.

Get started View on GitHub

Two interfaces, one workflow

Your dashboard

Review findings, update statuses, and track remediation progress in a clean web UI. Three themes — light, dark, and code — because you'll be spending time here.

Your agent's MCP server

Your AI coding assistant scans sites, triages issues, and triggers rescans through the Model Context Protocol. Works with Claude Code, OpenCode, and any MCP client.

Automated WCAG scanning

Browser-based scans powered by axe-core and Playwright. Configurable compliance profiles for WCAG 2.1 Level A, AA, and AAA.

High-Value Target grouping

Findings clustered by root cause so you fix the pattern, not just the symptom. Your agent can identify these clusters and start fixing while you review.

Diff-aware triage

See what's new, what's fixed, and what persists across scan runs. No more manually comparing reports — the tool tracks the delta for you.

Self-hosted, no strings

Runs on your own infrastructure with Docker Compose. No per-seat pricing, no vendor lock-in, no waiting on someone else's roadmap. MIT licensed.
